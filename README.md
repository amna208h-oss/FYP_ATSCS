# FYP_ATSCS
Final Year Project - AI Based Traffic Signal Control System 

This project aims to develop an AI-controlled traffic management system that prioritizes traffic based on real
time conditions and user needs. By dynamically adjusting traffic signals, the system seeks to reduce congestion, 
enhance safety, and promote efficient urban mobility,creating a more sustainable and equitable transportation 
network.
Problem Statement: This project addresses a critical issue in urban traffic management the inefficiency and 
inflexibility of traditional traffic signal control systems.
